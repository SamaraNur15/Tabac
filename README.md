# TabacWeb

Aplicación web funcional para un restaurante ubicado en San Salvador de Jujuy, Argentina. Desarrollada como un proyecto por Samara Nur Zamar.

## Tecnologías utilizadas

- **Backend:** Node.js + Express
- **Base de datos:** MongoDB
- **Frontend:** React
- **Autenticación:** JSON Web Tokens (JWT)
- **Estilos:** CSS
